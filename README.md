# رزومه HTML

این مخزن شامل تمرین ساخت رزومه با HTML است.

##  توضیحات تمرین
در این پروژه یک فایل `index.html` ساخته شده که شامل بخش‌های زیر است:
- نام و اطلاعات تماس  
-   معرفی کوتاه  
- تحصیلات  
- تجربه‌ها و پروژه‌ها  
- مهارت‌ها  


در این تمرین از تگ‌های معنایی HTML5 استفاده شده است و هیچ CSS یا JavaScript اضافه نشده.
